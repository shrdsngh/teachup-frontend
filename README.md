# Getting started

1. Run npm start to start the application. Make sure you are already running the backend on PORT: 8899.
2. On the first screen, you will see two links, Login and Register. Click on Login.

## Login as admin

ADMIN CREDENTIALS

### Admin Panel

1. You will see the list of instructors on the admin panel. Below the list, there is a section to add the course as required.
2. You can select the instructor and the date of the lecture before adding the course. You can select only those instructors who are available on that particular date.
